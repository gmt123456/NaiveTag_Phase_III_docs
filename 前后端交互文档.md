### 个人在一级任务中的参与

```
worker可以查看自己在以及任务中的参与状况情况
访问url: /worker/task/myparticipation.html

参数
token
taskState // 这儿要用户去选择

--> 这个参与情况类似于迭代二的任务状态，有等待结算，过期，进行中。 // waiting, expired, doing
--> 当这个一级任务结束后，所有参与的小任务只有已完成、过期两个状态。 // expired, finished
--> 已完成/过期/等待结算不能点进去看

如果这个一级任务还没有结束：

  {

  }
```
